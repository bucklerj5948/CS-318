COMPETITIVE RANKS

There are currently 23 different ranks in Rocket League. Each rank consists of 4 different divisions, in which you need to surpass all four in order to reach the next rank, so technically there are even more than 20 ranks if you count divisions, but that just makes it way too complicated. Ranks are calculated by MMR (match making rating), which is a number that goes up and//or down various values depending on if you won or lost the last match you played. The implementation of the ranking system definitely makes the game much more intense and engaging, as players are constantly trying to climb the rank ladder.
